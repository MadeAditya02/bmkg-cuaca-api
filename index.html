API Cuaca BMKG

Endpoint

  • /provinces (Mendapatkan data list provinsi Indonesia)
  • /districts?provinceId=bali (Mendapatkan data list kabupaten/kota berdasarkan id provinsi)
  • /cuaca?provinceId=bali&districtId=501162 (Mendapatkan data cuaca berdasarkan id provinsi dan id kabupaten/kota)

Note : Anda bisa menambahkan param receive pada endpoint cuaca untuk mendapatkan data yang spesifik, dengan value berupa key dari jenis data yang ingin didapatkan (sifatnya opsional)

(Contoh: /cuaca?provinceId=bali&districtId=501162&receive=humidity)

List key receive yang tersedia :

  • timestamp: Mendapatkan data timestamp
  • humidity: Mendapatkan data kelembapan udara
  • min_humidity: Mendapatkan data kelembapan udara maksimum
  • max_humidity: Mendapatkan data kelembapan udara minimum
  • temperature: Mendapatkan data suhu udara
  • max_temperature: Mendapatkan data suhu udara maksimum
  • min_temperature: Mendapatkan data suhu udara minimum
  • weather: Mendapatkan data cuaca
  • wind_direction: Mendapatkan data arah angin
  • wind_speed: Mendapatkan data kecepatan angin
